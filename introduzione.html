Istruzioni per la realizzazione della POC INCISO

Crea un'applicazione web proof-of-concept per il sistema INCISO (gestione invalidità civile) utilizzando Bootstrap Italia come framework CSS. L'applicazione deve simulare un sistema reale caricando dati da file JSON statici invece di chiamate API.

Framework e Tecnologie

  • CSS Framework: Bootstrap Italia (https://italia.github.io/bootstrap-italia/)
  • HTML5 semantico
  • JavaScript vanilla o framework leggero (React/Vue opzionale)
  • Dati: File JSON statici per simulare le risposte API

Struttura delle Pagine

1. PAGINA DI RICERCA (Homepage)

Layout:

  • Header con logo e titolo "Sistema INCISO - Gestione Invalidità Civile"
  • Barra di ricerca centrale prominente con:
    • Campo input per: Codice Fiscale, Cognome, Nome, Data di Nascita, o Codice Posizione (es. "S 400")
    • Pulsante "Cerca" (stile primario Bootstrap Italia)
    • Suggerimenti placeholder: "Cerca per CF, Nome, Cognome, Data Nascita o Posizione..."

Risultati della Ricerca:

  • Tabella responsive con colonne:
    • Codice Fiscale
    • Cognome e Nome
    • Data di Nascita
    • Data Decesso (se presente, evidenziata in rosso)
    • Rappresentante Legale (icona + nome se presente)
    • Tipologie Invalidità (badge colorati vivaci):
      • ■ Invalidità Civile (Blu #0066CC) - Con percentuale
      • ■ Cecità (Arancione #FFA500) - Con percentuale
      • ■ Sordità (Azzurro #00B4D8) - Con percentuale
  • Ogni riga cliccabile per aprire il fascicolo personale

3. PAGINA GESTIONE COMUNICAZIONI

Accesso: Da pulsante "Nuova Comunicazione" o "Genera Comunicazione" da Evidenza

Layout:

  • Form wizard a step:
    1. Selezione Template: Grid di card con modelli predefiniti (Avvio Procedimento, Richiesta Documentazione, Comunicazione Esito, etc.)
    2. Compilazione: Editor WYSIWYG simulato (textarea con formattazione base) pre-compilato con dati del template
    3. Revisione: Anteprima del documento generato (iframe o div formattato)
    4. Conferma:
      • Checkbox "Richiedi protocollazione"
      • Checkbox "Richiedi firma automatica"
      • Pulsante "Invia" (simula chiamata a IPROX)

Componenti UI Riutilizzabili Bootstrap Italia

  1. Header/Navbar: Logo + Titolo + Menu utente
  2. Breadcrumb: Navigazione contestuale
  3. Card: Per sezioni informative
  4. Accordion: Per fascicolo personale (alternativa a tab)
  5. Tab Navigation: Per fascicolo personale
  6. Form Controls: Input validati, date picker, dropdown
  7. Table responsive: Con ordinamento e paginazione
  8. Badge: Per stati e categorie
  9. Button Group: Per azioni multiple
  10. Modal: Per form e conferme
  11. Alert: Per messaggi sistema e presa in carico
  12. Progress bar: Per step wizard
  13. Timeline: Per note cronologiche

Funzionalità JavaScript

  • Ricerca: Filtro dinamico della tabella risultati
  • Caricamento JSON: Fetch API per simulare chiamate asincrone
  • Validazione Form: IBAN, CF, date
  • Ordinamento Tabelle: Click su header colonna
  • Filtri Avanzati: Applicazione multipla con aggiornamento dinamico
  • Export CSV: Generazione file da dati JSON
  • Local Storage: Simulazione presa in carico e stato sessione
  • Toast Notifications: Feedback operazioni (Bootstrap Italia)

Note Implementative

  • Usa data attributes per collegare azioni a elementi DOM
  • Implementa lazy loading per tab/accordion (carica JSON solo quando aperto)
  • Simula ritardi API con setTimeout per realismo (300-800ms)
  • Gestisci stati di errore con messaggi utente chiari
  • Responsive: Ottimizza tabelle per mobile (scroll orizzontale o card collassabili)
  • Accessibilità: ARIA labels, navigazione da tastiera, contrasto colori
